# TradingView Pine Scripts

Collection of Pine Script indicators and TradingView tools.

## 📌 Projects

### SR Hunter V4

**SR Hunter V4** is a Pine Script indicator built for TradingView.

The indicator focuses on layered market analysis:

- Trend Engine
- Structure Engine
- Location Engine
- Trigger Engine
- Quality Engine
- Risk Engine
- Decision Engine
- UI Output Panel

## 🎯 Purpose

SR Hunter is designed as a validation partner, not a signal machine.

The main idea:

> Think First. Trade Later.

## 🧰 Tech Stack

- Pine Script v6
- TradingView
- Technical analysis tools
